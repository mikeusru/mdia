Using the Software

What the software does


The Scanimage 3.8.1 Uncaging Automation software (We'll refer to the module as DIA, Dendrite Imaging Automation) allows the user to use Autofocusing, Drift Correction, and imaging and uncaging of multiple positions over a period of time.


Each available function is explained in this help file. You may refer to a detailed explanation of how to carry out each function, but it is also recommended that you familiarize yourself with the available controls in each window


Note: before trying anything with cells, try it with beads first. The software is buggy, and it's good to catch errors before they kill your samples.
