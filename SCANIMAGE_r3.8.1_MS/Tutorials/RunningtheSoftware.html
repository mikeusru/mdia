Start it up!


Once you set up the correct folders for MATLAB to work in, run scanimage as you previously would.


1. In the command line, type scanimage and press enter.


2. Once everything loads up, you're good to go. The Dendrite Imaging Automation window will allow you to access and control most of the automation parameters which you'll be using.


3. If you kept the option to automatically arrange your figures on in mdia_settings.m, you may have noticed that your windows tiled themselves once everything was done loading. If you prefer a different arrangement, then arrange the figures to your liking, select 'Options', 'Save Window Positions' in the Dendrite Imaging Automation window.
