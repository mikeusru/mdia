Advanced Timing Options





  • Stagger Uncaging
    • Toggle Stagger mode. When off, all positions are uncaged in sequence. When on, each position is uncaged exclusively, making sure that there is an appropriate amount of pre-uncaging and post-uncaging imaging time for all positions.



  • Exclusive Imaging
    • If stagger mode is on, this option allows imaging of only one position at a time both before and after uncaging. Set the duration to 0 to avoid this.


  • Initial Imaging
    • Initial imaging will cycle between and image the positions normally before the pre-uncaging process begins. For example, If you are doing staggered imaging and only wish to wait for 2 min between uncaging sessions but want to collect 10 min of imaging data before the first uncaging session, set the initial imaging to 8 min and pre-uncage duration to 2 min.


  • Display Expected Timeline
    • This button shows a preview of the experimental timeline given all the current settings and saved positions. Uncaging events are presumed to take 1 minute.
