Staggered Uncaging Options





  • Stagger Uncaging
    • Toggle Stagger mode. When off, all positions are uncaged in sequence. When on, each position is uncaged exclusively, making sure that there is an appropriate amount of pre-uncaging and post-uncaging imaging time for all positions.



  • Exclusive Imaging
    • If stagger mode is on, this option allows imaging of only one position at a time both before and after uncaging. Set the duration to 0 to avoid this.
